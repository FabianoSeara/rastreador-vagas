# Trilha de Candidaturas 🎯

Um rastreador de candidaturas de emprego construído em React. Permite registrar vagas, acompanhar o status de cada candidatura (Aplicado, Entrevista, Teste técnico, Oferta, Rejeitado) e visualizar estatísticas da busca por emprego, como taxa de resposta.

## ✨ Funcionalidades

- Cadastro de vagas com empresa, cargo, link, status, data e notas
- Atualização de status conforme o processo avança
- Busca por empresa ou cargo
- Filtro por status
- Painel com estatísticas: total de candidaturas, processos em andamento, ofertas recebidas e taxa de resposta
- Dados persistidos automaticamente

## 🛠️ Tecnologias

- React (hooks: `useState`, `useEffect`, `useMemo`)
- [lucide-react](https://lucide.dev/) para ícones
- CSS-in-JS (estilos inline)
